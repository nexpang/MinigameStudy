### 객체 설명
+ ▼: 유닛(적또는 아군 *색으로 구분)<br/>
	+ 유닛의 위치, 적, 아군 판단 변수, 움직임 구현<br/>
+ ▲: 플레이어<br/>
	+ 플레이어의 위치, 이동 구현<br/>
+ ☆: 총알(플레이어)<br/>
	+ 플레이어가 발사하는 총알 유닛과 같은 객체지만 변수로 다르게 움직임<br/>
+ GameManager: 게임메니져<br/>
	+ 게임의 UI, 맵 들을 그려주고, hp, score, highScore같은 변수와 게임의 거의 모든 시스템을 구현<br/>
